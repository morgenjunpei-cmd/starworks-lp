## ▌ フェーズ3：契約書/合意書で明記すべき条項

### 必須条項チェックリスト

- [ ] **制作範囲の明確化**（第2条）：何を作るか・何を作らないかの明確化
- [ ] **修正回数の上限**（第3条）：無料修正回数・超過時の費用
- [ ] **素材提供遅延時の対応**（第4条）：納期延長・一定期間後の解約権
- [ ] **検収期間・検収完了条件**（第5条）：納品後7営業日・異議なしで完了とみなす
- [ ] **著作権の帰属と利用許諾**（第6条）：制作物の権利は制作者に帰属・利用許諾の範囲
- [ ] **著作者人格権の不行使**（第7条）
- [ ] **AI・フリー素材の使用**（第8条）：商用利用確認済みの外部素材使用の旨
- [ ] **ポートフォリオ掲載許可**（第9条）：制作実績としての掲載許可
- [ ] **口コミ・体験談**（第10条）：強制しない旨・ステマ対応
- [ ] **外部サービス免責**（第11条）：ドメイン・サーバー等の外部起因トラブル
- [ ] **SEO・集客免責**（第12条）：検索順位・売上の保証なし
- [ ] **損害賠償上限**（第13条）：支払済み金額を上限とする
- [ ] **秘密保持**（第14条）：3年間の秘密保持義務
- [ ] **反社排除**（第15条）：反社会的勢力との関係がないことの表明
- [ ] **契約解除事由**（第16条）：解除できるケースの明確化
- [ ] **準拠法・管轄**（第17条）：日本法・仙台地裁を専属的合意管轄

## ▌ インボイス（適格請求書）について

> **現在の状況を確認してから以下を選択する：**

**ケース①：インボイス登録済みの場合**
- 請求書に「登録番号：T-XXXXXXXXXXXXXXX」を記載する
- 課税事業者のクライアントには必要

**ケース②：インボイス未登録（免税事業者）の場合**
- 「インボイス対応していません（免税事業者）」をFAQ・見積書に記載
- クライアントの経理担当者に事前確認を促す
