## ▌ セクション0-2：サンプル展示システム仕様（重要）

> 本テンプレートは **マーキーカルーセル + ライトボックス** による高度なサンプル展示を採用。

### マーキーカルーセルの仕様

- **機能**：9枚の制作サンプル画像が60秒周期で無限横スクロール
- **レイアウト**：PC 3列 / タブレット 2列 / スマホ 1列（動的変動）
- **CSS アニメーション**：`@keyframes marquee-scroll`（0% → 100% = 60秒で50%移動 = 無限ループ）
- **エッジマスク**：左右端に `mask-image: linear-gradient()` で徐々に透明化（ポリッシュ効果）
- **ホバー挙動**：PC でマウスホバー時は自動再生を一時停止
- **タッチ対応**：スマートフォンでスワイプ検出（50px以上の移動で次/前スライド）
- **画像重複**：DOMに18個の要素（9×2）。ただしライトボックスは9個のユニーク画像のみを表示

### ライトボックスの仕様

- **トリガー**：各サンプルカード（`<button class="sample-card" data-lightbox>`）をクリック
- **背景**：暗めのダークオーバーレイ（`rgba(8,24,48,.92)`）
- **ナビゲーション**：
  - ← → ボタン（左右移動）
  - Escape キー（閉じる）
  - 矢印キー（← / →）で次/前
  - スマートフォンでスワイプ対応（横スワイプで次/前）
- **画像の重複排除**：DOM の重複要素（マーキー用）から `Set` で unique な `data-src` のみ抽出
- **データ属性**：各 button に `data-src="path/to/image.png"` と `data-caption="説明文"` を必須

### HTMLマークアップ例

```html
<!-- マーキートラック内のサンプルカード -->
<button type="button" class="sample-card" data-lightbox 
        data-src="assets/images/samples/sample-01.png" 
        data-caption="クライアント名：ABC美容室">
  <img src="assets/images/samples/sample-01.png" alt="制作実績：ABC美容室">
</button>

<!-- ライトボックスHTML（mainの直後に1つだけ） -->
<div class="lightbox" id="lightbox" role="dialog" aria-modal="true" aria-hidden="true">
  <button type="button" class="lightbox-close" aria-label="閉じる">×</button>
  <button type="button" class="lightbox-nav lightbox-prev" aria-label="前の画像">‹</button>
  <button type="button" class="lightbox-nav lightbox-next" aria-label="次のスライド">›</button>
  <figure class="lightbox-figure">
    <img class="lightbox-img" alt="">
    <figcaption class="lightbox-caption"></figcaption>
  </figure>
</div>
```

### CSSクラス構成

| クラス名 | 用途 |
|---------|------|
| `.marquee` | マーキートラック外側コンテナ |
| `.marquee-track` | スクロールするトラック本体（animation 適用） |
| `.sample-card` | 個別カード（button要素） |
| `.lightbox` | ライトボックスコンテナ |
| `.lightbox.is-open` | ライトボックス表示時のクラス |
| `.lightbox-img` | ライトボックス内の画像 |
| `.lightbox-nav` | 左右のナビゲーションボタン |

### ⑦サンプル実績（マーキーカルーセル展示）

**構成要素：**
- 見出し：「制作実績」または「過去の制作事例」
- マーキーカルーセル：9枚のサンプル画像が60秒周期で無限スクロール
  - PC: 3列表示（同時に3枚見える）
  - タブレット: 2列表示
  - スマホ: 1列表示（順番に流れる）
- 各カード：クリック/タップで拡大表示（ライトボックス）
- ライトボックス機能：

  建築  = 工務店・塗装・リフォーム
  士業  = 税理士・社労士・行政書士
  制作  = ホームページ制作・デザイン

デザインタイプコード：
  A = シンプル・クリーン型（白基調）
  B = 高級感・ダーク型（黒・金基調）
  C = 温かみ・ナチュラル型（ベージュ・木目調）
  D = 信頼感・プロフェッショナル型（ネイビー・グレー）

例：
  LP-001_汎用_A  ← このファイル（テンプレートVer1.0）
  LP-002_美容_C  ← 美容室向けナチュラルデザイン
  HP-001_飲食_C  ← 飲食店向け複数ページ
```

### 新テンプレート追加時のルール

1. このファイル（Ver1.0）をコピーして新ファイルを作成する
2. 「テンプレート識別情報」のIDと業種・デザインタイプを変更する
3. ヒアリングシートの変数は基本共通。業種固有の項目は追記する
4. デザイントーンのカラーパレットとフォントを業種向けに変更する
5. 各セクションの構成メモを業種特性に合わせて調整する
6. 指示文サンプルは業種名を差し替えるだけで使える状態にする
7. テンプレート管理台帳.md に追記してIDを登録する
