# Star+ 法務チェックリスト Ver1.0

> 作成日：2026年5月18日
> 目的：制作物の公開前・クライアント対応前に確認すべき法務事項の一覧

---

## ▌ チェックリスト1：Star+ 自社サイト 公開前チェック

### プライバシーポリシー
- [ ] 取得する個人情報の種類が記載されているか
- [ ] 利用目的が明記されているか
- [ ] 第三者提供の有無と条件が記載されているか
- [ ] 外部サービス（Formspree / Googleアナリティクス / サーバー等）の利用が記載されているか
- [ ] 個人情報の保存期間・削除方針が記載されているか
- [ ] Cookie・アクセス解析・外部送信の説明が記載されているか（ブラウザでの拒否方法含む）
- [ ] 開示・訂正・利用停止・削除請求の対応が記載されているか
- [ ] お問い合わせ窓口（担当者名・メール）が記載されているか
- [ ] 制定日・最終改定日が記載されているか

### 特定商取引法ページ（law.html）
- [ ] 販売事業者名・運営責任者名が記載されているか
- [ ] 所在地が記載されているか（非公開の場合は「請求があれば開示」旨を記載）
- [ ] 電話番号・メールアドレスが記載されているか
- [ ] 販売価格（税込表記）が記載されているか
- [ ] 商品代金以外の必要料金（ドメイン・サーバー費等）が記載されているか
- [ ] 支払い方法・支払い時期が記載されているか
- [ ] サービス提供時期が記載されているか
- [ ] キャンセル・返金ポリシーが記載されているか（着手前・着手後・納品後で分けて記載）
- [ ] 動作環境が記載されているか
- [ ] 返品の可否が記載されているか

### お問い合わせフォーム
- [ ] 個人情報の利用目的の説明文（フォーム付近）が設置されているか
- [ ] プライバシーポリシーへのリンクが設置されているか
- [ ] 同意チェックボックス（必須）が設置されているか
- [ ] チェックなしでは送信できない制御が実装されているか

### サービスページ・FAQ
- [ ] 料金がすべて税込表記になっているか
- [ ] 「含まれないもの」（ドメイン・サーバー費等）が明記されているか
- [ ] SEO・集客・売上の保証をしていないことが明記されているか
- [ ] 追加費用は「事前の合意後に発生する」旨が明記されているか
- [ ] 保守サービスの解約条件が明記されているか

## ▌ チェックリスト2：クライアント案件 着手前チェック

### ヒアリング段階
- [ ] 業種を確認した（法規制が強い業種でないか）
  - 医療・クリニック → 医療広告ガイドライン確認必要
  - 美容・エステ → 薬機法・景品表示法確認必要
  - 整体・整骨院・マッサージ → 医療広告・誇大表現チェック必要
  - パーソナルジム → 薬機法（ダイエット効果の表現）チェック必要
  - 飲食 → 食品衛生・酒類広告等の確認

### 契約段階
- [ ] 制作範囲を書面（メールでも可）で明確に合意したか
- [ ] 修正回数を明記・合意したか
- [ ] 支払い条件（着手金・残金）を書面で確認したか
- [ ] 著作権の帰属・利用許諾について合意したか
- [ ] ポートフォリオ掲載の許可を得たか
- [ ] 口コミ・体験談について強制しない旨を伝えたか

## ▌ チェックリスト3：制作中チェック

### コンテンツの表現
- [ ] 以下の断定・保証表現を使っていないか
  - 「必ず集客できる」「必ず問い合わせが増える」
  - 「Googleで上位表示される」「検索1位になれる」
  - 「売上が上がる」「絶対に安心」
  - 「完全対応」「No.1」「最安値」「唯一」
- [ ] 代替表現を使用しているか
  - 「予約・問い合わせにつながりやすい導線を設計します」
  - 「SEO基本設定を行います（検索順位は保証できません）」
  - 「事前の合意なく追加費用を請求することはありません」

### 業種別の特別チェック（医療・美容・整体・ジム）
- [ ] 薬機法に抵触する表現がないか（効果・効能の断定表現）
  - NG例：「〇〇kg痩せる」「肌が若返る」「症状が治る」
  - OK例：「サポートします」「目指せます」「個人差があります」
- [ ] 景品表示法に抵触する表現がないか（優良誤認・有利誤認）
- [ ] 医療広告ガイドラインに抵触していないか（医院・クリニックの場合）
- [ ] 口コミ・体験談の掲載に顧客の許可を得ているか

### AI生成素材・フリー素材
- [ ] 使用する素材が商用利用可であることを確認したか
- [ ] AI生成ツールの利用規約で商用利用が認められているか確認したか（Midjourney・DALL-E・ChatGPT等、ツールによって異なる）
- [ ] フリー素材サイトの利用規約を確認したか（著作権表記が必要か等）

## ▌ チェックリスト4：GA4・GTM・Cookie 導入時チェック

- [ ] プライバシーポリシーにGA4/GTMの利用を記載しているか
- [ ] Cookieの利用とブラウザでの拒否方法を説明しているか
- [ ] Googleアナリティクスオプトアウトの手段を案内しているか
- [ ] リマーケティング・広告機能を使用する場合、別途明示しているか
- [ ] GA4のデータ保持期間を確認・設定したか（デフォルト14ヶ月）

---

## ▌ チェックリスト5：口コミ・体験談・SNS投稿 対応

- [ ] 口コミ投稿を強制・誘導していないか
- [ ] 「星5をお願いします」「好意的なレビューをお願いします」等の指示をしていないか
- [ ] モニター体験に基づく投稿の場合、「#PR」「#モニター」等の明示を促しているか（ステルスマーケティング規制対応）
- [ ] 口コミ・体験談をサイトに掲載する場合、本人の事前承諾を得ているか
- [ ] 掲載した体験談が実際の体験に基づいていることを確認できるか

## ▌ チェックリスト6：納品前 最終チェック

- [ ] 「○○」「XXX」等のプレースホルダーが残っていないか
- [ ] 料金が最新情報で税込表記になっているか
- [ ] プライバシーポリシーへのリンクが全ページに設置されているか
- [ ] 特定商取引法ページへのリンクがフッターに設置されているか
- [ ] フォームの同意チェックが必須になっているか
- [ ] SEO・売上等の保証表現がないか再確認する
- [ ] 誇大広告・優良誤認につながる表現がないか

## ▌ チェックリスト7：公開後・Google連携チェック

- [ ] SSL（https://）でアクセスできるか
- [ ] GA4でデータが受信されているか確認したか
- [ ] Googleサーチコンソールに登録したか
- [ ] Googleビジネスプロフィールに公開URLを登録したか（実店舗の場合）
- [ ] sitemap.xmlが送信済みか（サーチコンソールから）

## ▌ 注意：弁護士への確認が必要なケース

以下のケースは、必ず弁護士・行政書士等の専門家への確認を推奨する：

- 医療・クリニック・歯科・薬局向けサイトの制作
- 大規模な個人情報収集・管理システムの構築
- 業種固有の法規制が強い案件（風俗営業法・酒類・金融等）
- 契約トラブルが発生した場合
- 著作権侵害が疑われる場合
