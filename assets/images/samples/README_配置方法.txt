  ChatGPT Image 2026年5月16日 00_29_37 (2).png  →  sample-02.png
  ChatGPT Image 2026年5月16日 00_29_37 (3).png  →  sample-03.png
  ChatGPT Image 2026年5月16日 00_29_37 (4).png  →  sample-04.png
  ChatGPT Image 2026年5月16日 00_29_38 (5).png  →  sample-05.png

============================================
画像が未配置でも問題ありません
============================================
HTML 側で読み込みエラーを検知すると、自動で
「Sample Mockup」のプレースホルダーが表示される
仕様になっています。先に他の作業を進めても大丈夫です。

============================================
推奨スペック
============================================
- 形式：PNG / JPG / WebP（PNG 推奨）
- アスペクト比：16:10（横長）
- 解像度：1280×800 以上を推奨
- 1枚あたり：500KB 以内が望ましい（圧縮推奨）
