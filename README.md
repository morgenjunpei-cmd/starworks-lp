# STAR WORKS LP — 制作物ファイル一式

## 📂 フォルダ構成

```
STARWORKS_LP/
├── index.html          ← トップページ
├── service.html        ← サービス・料金
├── faq.html            ← よくある質問
├── contact.html        ← お問い合わせ
├── README.md           ← このファイル（プロンプト集つき）
└── assets/
    ├── style.css       ← 共通スタイル
    ├── script.js       ← 共通スクリプト
    └── images/
        ├── logo.png    ← STAR+ ロゴ
        ├── why-01.png  ← 「選ばれる理由」画像1（要・ChatGPT生成）
        ├── why-02.png  ← 「選ばれる理由」画像2（要・ChatGPT生成）
        └── why-03.png  ← 「選ばれる理由」画像3（要・ChatGPT生成）
```

## 🎨 デザイン仕様

- **配色**：ダークネイビー `#0D2447` / ティール `#1FB3A8` / ゴールド `#E8C547`（ロゴ由来）
- **レイアウト**：PC＝左サイドバー固定 / スマホ＝右上ハンバーガーメニュー
- **構成**：4ページ（トップ／サービス／FAQ／お問い合わせ）
- **アニメーション**：スクロールで要素がふわっと下から現れる（フェードアップ）

## 🖼️ ChatGPTで画像を生成するプロンプト

「選ばれる3つの理由」セクションの画像3枚を、ChatGPT（GPT-4o）でDALL·Eに生成させるためのプロンプトです。

### 共通スタイル
3枚とも統一感を出すため、**スタイル指示は3枚で同じ**にしてあります。
1:1の正方形画像、800×800px以上、透明背景PNG推奨。

### 🔧 生成した画像の使い方

1. 生成した画像を `why-01.png` / `why-02.png` / `why-03.png` という名前で保存
2. `assets/images/` フォルダにドラッグ＆ドロップ
3. `index.html` の該当箇所のコメントアウトを外す
   （`<!-- <img src="..."> -->` の `<!--` と `-->` を削除）
4. ブラウザで `index.html` を開いて確認

## ✉️ お問い合わせフォームの設定（Formspree）

お問い合わせフォームをGmailに転送するための設定手順です。

1. https://formspree.io/ にアクセス
2. 無料アカウントを作成（Googleアカウントでログイン可）
3. 「New Form」をクリック
4. 送信先メールアドレスに `starworks.shoji@gmail.com` を設定
5. 作成されたフォームID（例：`xpwzabcd`）をコピー
6. `contact.html` を開き、`YOUR_FORM_ID` を取得したIDに置き換え

```html
<!-- 修正前 -->
<form action="https://formspree.io/f/YOUR_FORM_ID" method="POST">

<!-- 修正後（例） -->
<form action="https://formspree.io/f/xpwzabcd" method="POST">
```

これでフォーム送信内容がGmailに届きます。
無料プランは月50件まで送信可能。

## 📝 修正したい時のChatGPTプロンプト雛形

サイトの文言や色を変えたい時は、以下のテンプレートを使うとClaudeに正確に伝えられます。

### 文言を変えたい時

```
STARWORKS_LPのトップページ（index.html）にある
「想いを、Webでカタチに。」というキャッチコピーを
「〇〇〇〇」に変更してください。
```

### 色を変えたい時

```
STARWORKS_LPの assets/style.css の中の
--teal （#1FB3A8）を #〇〇〇〇〇〇 に変更してください。
他のCSSルールはそのまま維持してください。
```

### プランの内容を変えたい時

```
STARWORKS_LPの service.html のスタンダードプランの
価格を 88,000円から 〇〇〇〇〇円 に変更し、
特徴リストの「修正 3回まで」を「修正 4回まで」に変更してください。
```
